À propos du scheduler
Exécution automatique

Les tâches planifiées sont exécutées automatiquement par un process séparé (scheduler). Par défaut, les rapports hebdomadaires sont envoyés le lundi à 8h00 et le nettoyage est effectué le dimanche à 3h00.

Configuration

Les horaires peuvent être modifiés via les variables d'environnement : SCHEDULER_WEEKLY_REPORT_DAY, SCHEDULER_WEEKLY_REPORT_HOUR, etc. Consultez la documentation pour plus de détails.
